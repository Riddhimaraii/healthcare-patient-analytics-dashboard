# Healthcare Patient Analytics & Decision Support Dashboard

## Project Overview
This project analyzes hospital readmission data to identify patient trends, cost drivers, and factors influencing readmissions.

## Dataset
Hospital Readmissions Dataset (Kaggle)

## Tools Used
Python (Pandas, Matplotlib), SQL, Excel, Power BI

## Key Work
- Cleaned and analyzed healthcare datasets
- Performed exploratory data analysis on patient demographics
- Built dashboards to track healthcare KPIs
- Generated insights to support data-driven decisions
